## Player assets

`miner-custom-sheet.png` is a project-local player sheet drawn specifically for Minerador Arqueologico.

It uses 12 frames at 80x80 pixels:

- frame 0: idle
- frames 1-6: walk
- frames 7-11: mining swing

The pickaxe is part of the character frames so the miner actually holds it during idle, walk, and mining.
